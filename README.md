# Bank ATM System

A Python-based ATM simulation with PIN authentication and basic operations:
balance inquiry, cash withdrawal, deposit, and mini-statement.

## Features
- PIN authentication
- Balance inquiry
- Cash withdrawal with validation (insufficient balance check)
- Cash deposit
- Optional mini-statement (last few transactions)

## Tech stack
- Python (OOP, file handling)
- Optional: CSV for simple data storage
